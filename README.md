# 🐟 ワンタッチすいぞくかん

こどもが描いたお魚を写真でとって、大きな海（テレビ）で泳がせる PWA アプリです。
2台の iPad（①海＝テレビ用 / ②カメラ＝撮影用）で使います。

---

## 今回の改修内容

### 1. PWA化（iPadにインストールして使える）
- `manifest.json` / `sw.js`（Service Worker）/ アイコン一式を追加
- iOS用メタタグ（ホーム画面追加・全画面表示）
- **画面スリープ防止**（海モード中はテレビ表示が消えない / iPadOS 16.4+）

### 2. 接続まわりの安定化
- 海モードの **番号かぶり（ID衝突）を自動で別番号に作り直し**
- 信号サーバーが切れても **自動で再接続**
- **信頼性チャネル**（reliable）で画像を確実に送信
- 海モードに **接続台数・お魚の数の表示**
- カメラ側のエラー文言を状況別に表示（番号ちがい / ネット未接続）
- タイムアウトを3秒→8秒に延長（PWA起動直後の遅延対策）
- 送信前に **画像を自動縮小（長辺640px）** して通信を高速化＆テレビ側を軽量化

### 3. 切り抜き（クロマキー）精度の改善
- エッジを **なめらかに抜く（フェザリング）**
- 魚のフチに残る **緑色を消す（デスピル）**

### 4. 演出の強化（海モード）
- お魚が **画面の端から登場 → ぷるんと出てくる（ポップイン）**
- やわらかな上下のゆらぎ・傾き
- **光のカーテン（god rays）／海底の砂／揺れる海藻／到着リップル／効果音**
- 表示上限60匹（古い魚から自動退場で重くならない）

## 使い方（当日の流れ）

1. **①テレビ用iPad** … アプリを開いて「📺 おおきなうみ」をタップ → 画面左上に **4桁の番号** が出る。iPad を HDMI か AirPlay でテレビにつなぐ。
2. **②撮影用iPad** … 「📸 カメラ」をタップ → ①の番号を入力 → 「つながる！」
3. 緑の背景の前でお魚の絵を撮影 → スライダーで背景を消す → 「うみにおくる！」
4. テレビの海にお魚が泳ぎだす🐟

> 2台は **同じ Wi-Fi** につないでください（安定して接続できます）。

---

## デプロイ手順（HTTPSが必須）

カメラ機能（getUserMedia）と PWA インストールには **HTTPS配信** が必要です。
GitHub Pages が無料で簡単です。

### A. GitHubにファイルを置く
このフォルダの中身（`index.html` / `manifest.json` / `sw.js` / `icons/`）を
リポジトリ `kumApp39/suizokukan` に置きます。

- Gitが使えるなら：
  ```
  git add .
  git commit -m "PWA化＋接続安定化＋切り抜き改善＋演出強化"
  git push
  ```
- Gitが無いなら：GitHubのリポジトリ画面で **Add file → Upload files** から
  4つ（フォルダごと）をドラッグして commit でもOK。

### B. GitHub Pages を有効化
リポジトリの **Settings → Pages → Branch: main / (root)** を選んで保存。
数十秒後、`https://kumapp39.github.io/suizokukan/` で公開されます。

### C. iPadにインストール
1. Safari で公開URLを開く
2. 共有ボタン → **「ホーム画面に追加」**
3. ホーム画面のアイコンから起動（全画面のアプリとして動きます）

---

## 注意点
- iPad同士が**別ネットワーク**だと接続できないことがあります（TURNサーバー未設定のため）。同じWi-Fiを推奨。
- 初回はカメラの利用許可を「許可」してください。
- 緑背景はムラなく明るい緑布だと、いちばんキレイに抜けます。
